Description

This change involves permanently decommissioning a Linux virtual machine and its associated Azure resources. The server is no longer required for business operations, is not joined to Active Directory, and has no current application dependencies. The VM is currently included in Cohesity backups. The change will remove the server, its disks, NICs, public/private IPs (if any), and backup entries from the portal to reduce unnecessary cloud spend and improve environment hygiene.

Justification

The Linux server is no longer required for any business, operational, or application function. It is consuming Azure compute, storage, and backup capacity, resulting in unnecessary cost. Decommissioning the server will remove unused infrastructure, ensure an accurate inventory of active systems, and improve overall environment management.
There are no dependencies tied to this VM, and it is not joined to Active Directory. The responsible team has confirmed that all data or configurations on the server are no longer needed.
